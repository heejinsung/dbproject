## Features
- 데이터베이스 관리 시스템인 mariaDB 사용
- 로그인,로그아웃 / 회원가입 기능
- ADMIN 기능으로 회원관리,qna 게시글관리,메시지관리 등 전반적인 사이트 관리자기능 구현
- 유저레벨에 따라 관리자 권한 부여, 로그인시 관리자유저레벨만 admin 계정 접속가능
- 데이터 입,출력 및 검색 가능한 백엔드기능 구현

## 🛠Tech Stack
- PHP
- CSS
- JavaScript
- jQuery
